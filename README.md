# StackTutor

StackTutor is an AI-powered code review trainer for students learning to debug, understand errors, generate test cases, and improve code quality.

## Tagline

Learn to debug. Don't just get the answer.

## Platforms

- iOS
- Android
- Web landing page

## Tech Stack

- React Native
- Expo
- TypeScript
- Next.js
- Tailwind CSS
- Supabase
- Server-side AI APIs

## Product Positioning

StackTutor is not a homework cheating app. It is a guided debugging and code review trainer that helps students understand their mistakes through hints, explanations, test cases, and structured feedback.
