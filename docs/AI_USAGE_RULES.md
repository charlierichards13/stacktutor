# AI Usage Rules

StackTutor uses AI tools as engineering assistants, not as uncontrolled code generators.

## Rules

- Features are planned before implementation.
- AI-generated code must be reviewed before commit.
- No secrets or environment files may be exposed.
- Prompts must be scoped to small, reviewable tasks.
- UI changes must follow the design system.
- Every major feature should have acceptance criteria.
- AI output should be refactored into clean, maintainable TypeScript.

## Goal

The goal is to use AI to speed up implementation while preserving code quality, product consistency, and security.
