# StackTutor Product Spec

## One-liner

StackTutor is an AI-powered code review trainer that helps CS students debug code, understand errors, generate test cases, and learn programming concepts without simply being handed answers.

## Target Users

- Intro CS students
- Data structures students
- Interview prep students
- Self-taught programmers
- Interview prep students

## Core Problem

Students often get stuck debugging code and either waste hours guessing or use AI tools that give direct answers without teaching the reasoning.

## Product Promise

StackTutor gives structured, educational code feedback with hints, explanations, test cases, complexity analysis, and security-focused review modes.

## MVP Features

- User authentication
- Submit code
- Select programming language
- Select review mode
- Receive structured AI feedback
- Save review history
- View past reviews
- Delete reviews

## Supported Languages V1

- Java
- Python
- C++
- TypeScript

## Review Modes V1

- Find Bugs
- Explain Code
- Generate Test Cases
- Check Complexity
- Security Review
- Hint Mode

## Non-goals for V1

- Payments
- Classroom management
- Live collaboration
- Full IDE replacement
- Running arbitrary code on our servers
